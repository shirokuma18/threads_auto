## 🚨 スパム判定を避けるルール（必読）

### ❌ 絶対にやってはいけないこと

Threadsのスパム判定は厳しいです。以下は**確実にスパム認定**されます：

#### 1. **数字の羅列**
```
NG例:
「コンビニ代19,500円、カフェ代18,000円、サブスク4,500円...
 合計34,500円、年間414,000円の節約！」
```
→ 情報商材・広告に見える

**対策:**
- 数字は1-2個まで
- 「月2万円使ってた」程度に留める
- 細かい金額は書かない

#### 2. **Before/After形式**
```
NG例:
「【Before】
 コンビニ代：月19,500円
 貯金：0円

 【After】
 コンビニ代：月4,000円
 節約額：31,000円」
```
→ 広告の典型的フォーマット

**対策:**
- Before/Afterを明記しない
- 自然な文章で変化を語る
- 「3ヶ月前は...今は...」程度

#### 3. **箇条書きの多用**
```
NG例:
「【やったこと】
 ・方法1
 ・方法2
 ・方法3
 ・方法4
 ・方法5
 ...（10個以上）」
```
→ ノウハウ販売に見える

**対策:**
- 箇条書きは3個まで
- 自然な文章で書く
- リスト形式を控える

#### 4. **結果誘導フレーズ**
```
NG例:
「2ヶ月経った今：」
「半年後：」
「結果：」
```
→ 成功体験の売り込みに見える

**対策:**
- 「今」「最近」程度にぼかす
- 進行形で書く（「変わってきた」）
- 完成形を避ける

#### 5. **人間味の不足**
```
NG例:
感情表現なし
迷いなし
失敗なし
全部成功
```
→ AIや広告に見える

**対策:**
- 感情を入れる（「めっちゃ」「やばい」「嬉しい」）
- 迷いを書く（「続けられるかな」）
- 失敗も書く（「1日寝坊した」）
- 不安を正直に（「不安だけど」）

## ✅ バランス型投稿構成

### 3つのタイプをバランス良く

**1. リアルタイム試行錯誤（30%）**
- 「今まさに挑戦中」の記録
- 3日目、5日目、1週間目など進捗
- 「続けられるかな」「どうなるか楽しみ」
- 未完成・進行中の状態

例:
```
コンビニ行かないチャレンジ、3日目。

帰り道にコンビニあるから、
めっちゃ寄りたくなる。

「ちょっとだけ...」って誘惑がすごい。

でも先月、コンビニだけで2万円使ってた。
それ知ってから怖くなった。

とりあえず1週間頑張る。
続けられるかな。
```

**2. 体験談ベースノウハウ（50%）**
- 個人の物語として語る
- 感情・葛藤のプロセスを含む
- 「私も最初は...」「半年前の私...」
- 完成しすぎない（今も継続中など）

例:
```
半年前の私、完全に恋愛依存だった。

彼から連絡こないと不安で、
予定も全部彼優先。
友達との約束もドタキャン。

友人「それ、依存してるよ」

その言葉でハッとした。

で、少しずつ変えてみた。
・週1は友達と会う
・連絡は我慢（1日返信しない）
・趣味再開（ヨガ教室）

3ヶ月経った今：
不安が減った。
彼との関係も良くなった。

依存しない方が、愛される。
```

**3. 役立つノウハウ（20%）**
- 体系的な情報価値
- 保存・シェアされやすい
- ただし「私の経験から」という文脈
- 教科書的にならないよう注意

例:
```
固定費、見直してみた。

スマホ代8,000円→格安SIMで2,000円。
サブスク4,500円→1,000円に整理。

月2万円くらい浮きそう。

「こんなに払ってたんだ...」

見直すだけでこんなに変わるんだね。
```

```
[チャレンジ名]、[X]日目。
```

```
[時期]の私、[問題]だった。

[具体的な状況]

[気づき]

で、少しずつ変えてみた。
・[変化1]（シンプルに）
・[変化2]
・[変化3]

[現在の状況]

[学び・結論]
```

### テンプレート3: 気づき共有

```
[トピック]、[やってみた/気づいた]。

[Before状態]

「[心の声]」

[変化・発見]

[現在の状況]

[シンプルな結論]
```

## 🎯 文字数と構成

### 理想的な文字数

**リアルタイム挑戦記録**: 150-250文字
- 短く、リアルタイム感
- 感情中心
- 箇条書き最小限

**体験談ベース**: 250-400文字
- ストーリー性
- プロセス重視
- 箇条書き0-3個

**役立つノウハウ**: 150-300文字
- 簡潔に
- 具体例1-2個
- 箇条書き0-3個

### 絶対守るべきこと

- 数字: 1-2個まで
- 箇条書き: 0-3個まで
- 感情表現: 必ず入れる
- 「？」「！」: 必ず使う
- Before/After: 明記しない
