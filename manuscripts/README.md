# manuscripts/

ここには“まず小説を完成させ、それから投稿へ分割する”ための原稿を置きます。

命名規則（番号管理）
- `001_タイトル.md` のように通し番号で管理（2桁/3桁を推奨）
- 同日に複数作品を出す場合も、番号だけ前へ進める（日時に依存しない）

- 視点: 一人称（教師）。叙述は常体、会話は丁寧語。
- 構成: 朝→昼→放課後→夜（章見出しで区切り）。のちに24投稿等へ分割しやすいよう、段落末にゆるい余韻を置く。
- 倫理: 個人特定情報の回避／家庭事情の断定を避け、観察に根拠を添える。
