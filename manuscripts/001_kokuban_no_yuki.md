# 黒板の雪

（朝）
昇降口の冷気が、まだ湿った黒板を撫でる。袖に白い粉がうっすらつく。私は板書の最初の行を、いつもより少し下に下げた。上に空を残したい。そこに、今日の最初の“点”が置かれるかもしれないから。

かいとが早く来る。転校して二週間。彼はチョーク置きから白を一本だけ取って、黒板の隅にそっと点を打つ。「音、しませんね」と彼。軽く触れれば、たしかに鳴らない。私はうなずく。音がないまま、教室の空気が半歩だけ動いた気がする。

「ここ、始まりにしてもいいですか」
「いいよ。いちばん端からで大丈夫」

始まりの位置はいつも少し迷う。今日は、ここにする。

（午前）
休み時間、かいとは黒板消しの角で点の縁をそっとなぞる。粉がほどけて、小さな光が浮かぶ。私は白で点を二つ、細い線で結んでみせる。「ここから歩けるかな」と黒板に向かって訊ねるふりをする。彼は小さくうなずき、もう一つ点を足す。星座に見える。でも道にも見える。どちらでも困らない。

「消すと、怒られますか」
「消し方がやさしければ大丈夫。跡が残るくらいがいい」

消す技術は、作るのと同じくらい要る。私はそう思っている。

昼の光が窓枠で曲がり、ノートの罫線が波打って見える。黒板の丸に視線が集まるが、問いは出さない。触れない距離を、みんなが確かめている。私は急がない。問いは、待てば中から出てくる。

給食の匂いが廊下をゆっくり通る。「どこから降ってきたことにします？」と彼。「屋上から……は変ですか」「変でいい。想像の入口は多いほうが、帰り道も増える」。

（午後）
図工室から名札ケースを借りる。黒板の脇に小さな棚を作って並べる。「宛名を書けば、ここに手紙が住めます」。説明は短く。見えるほうを先に置く。かいとは線の先に点を打つ。「家まで行けるかな」「行けるように、曲がり角に印を置こう」。線の向きが、もう答えになっている。

チャイム。廊下の自転車のチェーンが一度鳴る。私は黒板の上にだけ雪を残す。教卓には降らせない。「先生の場所」と決めたくない。だれでも座れる余白を、必ず作っておきたい。

連絡帳に短く書く。「合図は選べます。鳴らす／鳴らさない、預ける／手元に置く」。書きすぎない。あとは見る。保健室からの伝言で、昼前に母から電話があったと知る。夜勤が続くらしい。夜の家の静けさを想像する。鳴らしたくない夜があるのは、ふつうのことだ。

（放課後）
かいとが黒板消しを持つ。私は何も言わず、端から撫でる速さだけ見せる。「この速さで、こう」。雪を踏むみたいに。跡が残るくらいがいい。半分だけ残った。残し方は、彼が決めた。私は受け取る役に回る。判断を私が持ち帰らないために。

保護者アプリに下書きだけ保存する。「鈴は安心の道具。鳴らす／鳴らさないは本人が選ぶ」。送信はしない。言葉の出番は、だいたい後ろから来る。上履き袋を肩にかけたかいとが、「先生、今日の丸、好きでした」と言う。「私も。明日はもう一つ増やそう」。声は短く、指先ははっきり動く。

教室の時計は夜の入り口でゆっくりになる。待ち合わせの点を、私は小さく光で縁取る。消し忘れ、という名前の配慮。窓を少し開ける。黒が深く戻る。白は浮いて見える。消さない理由が、今日の終わりになる。

（夜）
私はノートに今日の地図を写す。点、線、待ち合わせ、未命名の場所。名付けは急がない。名前は行動のあとでいい。職員室。電話を一本かける。出なかった。夜勤の家。鳴らさない夜。鳴らさない合図も、合図として機能するはずだ。

明日の板書を下書きする。いちばん上は空けておく。最初の雪は、だれにも踏ませない。冬の入口みたいに。蛍光灯を一本ずつ消す。黒が戻る。白い最初の点はそのまま。終わりは最初の向こうに置いて帰る。明日の朝の黒で、また確かめよう。彼がどこに点を足すのかを。

（あとがき／先生の声）
守りたかったのは、かいとの速さ。最初の白い点は「ここからでもいいよ」という合図にしたかった。だから、説明を先に並べず、点や線や待ち合わせみたいな“見える目印”を先に置いた。鈴は安心の道具。夜勤のある家では、鳴らさない夜がある。鳴らさないことも合図にできるように、学校の側に置き場所を作っておきたかった。丸や名札の棚は、そのための目印。

「半分だけ残す」は、上手く消せたかより、どう残したいと思えたかを大切にしたかったから。残し方は彼の手に残しておく。最初の点を最後まで残したのは、はじまりを踏ませないため。あれは勇気の入口。明日の朝の黒で、また一緒に確かめたい。
